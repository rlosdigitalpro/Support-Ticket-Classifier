# Support Ticket Classification Simulator (Python)

## Overview

This project simulates a simplified support ticket classification workflow similar to those used in structured technical support environments.

The system:

- Accepts ticket descriptions
- Classifies issues by category
- Assigns priority level
- Suggests initial action

---

## Purpose

Demonstrates:

- Logical decision-making
- SLA awareness
- Ticket triage thinking
- Structured support workflow design
